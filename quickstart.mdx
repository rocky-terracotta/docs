## 1. Generate an API key

<Steps>
  <Step title="Sign in to casebyte.ai">
    Go to [casebyte.ai/dashboard](https://casebyte.ai/dashboard) and sign in
    with your Casebyte account. API access is currently in private beta — if
    you don't see the **API keys** section, [email us](mailto:support@casebyte.ai)
    for access.
  </Step>
  <Step title="Open the API keys section">
    From your dashboard, open **Settings** and scroll to **API keys**.
  </Step>
  <Step title="Create a key">
    Click **New API key**, give it a descriptive name (e.g. `local-dev` or
    `production-worker`), and click **Create**.

    <Warning>
      You'll see the full key value **once**. Copy it now and store it in a
      secret manager. Casebyte only ever stores a hash of the key, so we cannot
      show it to you again. If you lose it, revoke it and issue a new one.
    </Warning>
  </Step>
</Steps>

API keys look like this:

```
cb_live_8f2a3b4c5d6e7f1a9b8c7d6e5f4a3b2c
```

The `cb_live_` prefix is part of the key — include it in your `Authorization`
header.

## 2. Make your first request

Search for the phrase **"wednesbury unreasonable"** across all of HK case law,
returning at most 5 results:

<CodeGroup>

```bash cURL
curl -G https://api.casebyte.ai/v1/cases/search \
  -H "Authorization: Bearer $CASEBYTE_API_KEY" \
  --data-urlencode 'q="wednesbury unreasonable"' \
  --data-urlencode 'limit=5'
```

```typescript Node.js
const res = await fetch(
  "https://api.casebyte.ai/v1/cases/search?" +
    new URLSearchParams({
      q: '"wednesbury unreasonable"',
      limit: "5",
    }),
  {
    headers: {
      Authorization: `Bearer ${process.env.CASEBYTE_API_KEY}`,
    },
  },
);
const data = await res.json();
console.log(data.results);
```

```python Python
import os
import requests

res = requests.get(
    "https://api.casebyte.ai/v1/cases/search",
    headers={"Authorization": f"Bearer {os.environ['CASEBYTE_API_KEY']}"},
    params={"q": '"wednesbury unreasonable"', "limit": 5},
)
res.raise_for_status()
print(res.json()["results"])
```

```go Go
req, _ := http.NewRequest("GET", "https://api.casebyte.ai/v1/cases/search", nil)
req.Header.Set("Authorization", "Bearer "+os.Getenv("CASEBYTE_API_KEY"))

q := req.URL.Query()
q.Set("q", `"wednesbury unreasonable"`)
q.Set("limit", "5")
req.URL.RawQuery = q.Encode()

res, _ := http.DefaultClient.Do(req)
defer res.Body.Close()
body, _ := io.ReadAll(res.Body)
fmt.Println(string(body))
```

</CodeGroup>

## 3. Retrieve the full judgment

Take a `case_id` from the search results and fetch the judgment text:

```bash
curl https://api.casebyte.ai/v1/cases/HKCFA_2021_12 \
  -H "Authorization: Bearer $CASEBYTE_API_KEY"
```

The response includes metadata plus the full plain-text judgment:

```json
{
  "case_id": "HKCFA_2021_12",
  "case_name": "Kwok Cheuk Kin v Director of Lands",
  "court": "HKCFA",
  "neutral_citation": "[2021] HKCFA 12",
  "action_numbers": ["FACV 4/2020"],
  "publish_date": "2021-04-21",
  "case_url": "https://www.hklii.hk/...",
  "doc_text": "JUDGMENT...",
  "matched_by": "case_id"
}
```

## 4. See who cites it

Find later judgments that cite this case:

```bash
curl https://api.casebyte.ai/v1/cases/HKCFA_2021_12/cited-by \
  -H "Authorization: Bearer $CASEBYTE_API_KEY"
```

```json
{
  "case_id": "HKCFA_2021_12",
  "total": 8,
  "citing_cases": [
    {
      "case_id": "HKCA_2023_487",
      "case_name": "...",
      "neutral_citation": "[2023] HKCA 487",
      "publish_date": "2023-09-15",
      "citation_text": "[2021] HKCFA 12"
    }
  ]
}
```
