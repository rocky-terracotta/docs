The Casebyte API gives you programmatic access to the same Hong Kong case-law
corpus that powers [casebyte.ai](https://casebyte.ai) — roughly 157,000
judgments from the HKSAR courts, with full text, neutral citations, and
case-to-case citation links.

It's a small, opinionated REST API. Three endpoints, one auth scheme, JSON in
and out.

## What you can do

<AccordionGroup>
  <Accordion title="Run boolean searches over the full corpus" icon="magnifying-glass">
    Search case names and judgment text with phrases, AND/OR/NOT operators,
    field-scoped queries, and fuzzy matching. Filter by court and date range,
    paginate with offset/limit.

    ```bash
    GET /v1/cases/search?q=%22wednesbury+unreasonable%22&court=HKCFA,HKCA
    ```
  </Accordion>
  <Accordion title="Retrieve a judgment by any identifier" icon="file-lines">
    Look up a case by canonical case ID, neutral citation, or action number.
    Returns metadata plus the full plain-text judgment.

    ```bash
    GET /v1/cases/HKCFA_2023_5
    GET /v1/cases/%5B2023%5D%20HKCFA%205
    GET /v1/cases/FACV%205%2F2022
    ```
  </Accordion>
  <Accordion title="Find cases that cite a judgment" icon="diagram-project">
    Returns later judgments that cite the case, deduplicated and sorted by
    publication date. Includes both direct case-to-case citations and citations
    by action number.

    ```bash
    GET /v1/cases/HKCFA_2023_5/cited-by
    ```
  </Accordion>
</AccordionGroup>

## Base URL

```
https://api.casebyte.ai
```

All endpoints live under `/v1/`. The unauthenticated `/health` endpoint returns
`{ "status": "ok" }` and is suitable for uptime checks.

## Authentication

Every request to `/v1/*` must carry a Bearer token:

```http
Authorization: Bearer cb_live_...
```

API keys are issued from your [casebyte.ai settings](https://casebyte.ai/settings)
page. See [Quickstart](/quickstart) for the full walkthrough.

## Coverage

| Field | Value |
|---|---|
| Jurisdiction | Hong Kong (HKSAR) |
| Courts | CFA, CA, CFI, DC, LT, FC, MC and tribunals |
| Approximate corpus size | ~157,000 judgments |
| Earliest judgment | 1946 |
| Update cadence | Daily, with new judgments typically available within 24 hours of publication |

UK and other Commonwealth jurisdictions are not exposed in v1.

## Rate limits

10 requests per minute, per API key. Responses include `X-RateLimit-*` headers
so you can pace yourself; over-limit requests return `429` with a `Retry-After`
header. Full details in [Rate limits](/guides/rate-limits).

## Need more?

If you're hitting rate limits, need bulk export, or want jurisdictions outside
